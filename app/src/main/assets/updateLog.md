# WenwenToMe 更新日志

## 2.0.1

* 浏览器找书补齐优化阅读、目录跳章、保存成书与统一缓存主链
* 浏览器优化阅读书进入统一阅读器后，支持整本缓存范围规划与缓存统计
* 阅读页补上打开书籍、关闭书籍的正式转场动画
* 发现页、详情页与 AI Hub 入口文案收口为正式版本口径
* GitHub Actions 云端 `android-ci` 再次验证通过，可继续作为主编译通道

## 2.0.0

* 正式安装包身份切回 `com.wenwentome.reader`
* 阅读内核切换为基于 Legado 的成熟底座
* 本地 TXT / EPUB、网文搜索与书源能力改走新主链路
* 项目仓库、发布入口与后续扩展能力继续保留在 WenwenToMe

## 迁移说明

* WenwenToMe 2.x 是底座重建版本，旧 1.x 版本已冻结保留
* 当前代码基于 Legado 底座演进并遵循 GPL-3.0
* 如升级前仍在使用旧版，请先自行做好备份
* 发布页与源码仓库：<https://github.com/yuelangmanle/wenwentome1>
