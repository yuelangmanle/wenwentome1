# WenwenToMe 1.x 旧线冻结说明

## 结论

- `1.x` 产品线冻结在 `v1.6`
- 旧线继续保留 Git 历史、tag、release 与 APK 下载入口
- 旧线不再继续增强自研 TXT / EPUB / 书源 / 阅读器内核

## 冻结原因

当前 1.x 的本地书解析、EPUB 可读性、阅读器布局和网文书源链路已经证明继续小修小补的收益很低。项目决定直接切换到更成熟的底座，而不是继续重复造轮子。

## 后续路线

- 新产品线从 `2.0.0` 开始
- 底座迁移目标为 `Legado`
- 仓库、品牌、包名、云端发版链路继续沿用 WenwenToMe
- WenwenToMe 的 AI Hub 与 GitHub 同步能力将在新底座上重新接回

## 回退说明

如果迁移中的新版本不稳定，用户可以继续下载并安装 `v1.6` 作为回退版本。该版本是旧线的最终稳定锚点。
