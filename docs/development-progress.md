## 当前风险

- `1.2` 正式版已发布，以下风险均为后续迭代项，不阻塞当前版本交付
- app 级回归仍以 `WEB` 主链为主，`TXT/EPUB` 的端到端烟雾覆盖仍可继续补宽
- PDF 阅读仍未实现，这一项按现阶段规划继续后置到 `1.3+`
- Release 签名沿用仓库现有 GitHub secrets，暂未做签名轮换，以避免升级兼容风险
- GitHub Actions 依赖链可继续做一次现代化整理，例如跟进运行时版本升级与缓存策略优化
