## 当前风险

- 本机仍缺少 Java 运行时，Gradle 验证统一卡在 `Unable to locate a Java Runtime`
- 当前工作树对应的 `android-ci` 结果仍待本次推送后的新 run 确认
- `android-release` 只接受 `main` 上的正式 tag，当前 `1.2` 还未进入正式发版
- GitHub artifact 下载到本地仍偶发 `EOF`，当前确认更像下载链路问题，不是构建失败
- app 级回归仍以 `WEB` 主链为主，本地 `TXT/EPUB` 的 app 级烟雾覆盖还不够宽
- PDF 阅读仍未实现，这一项按当前阶段规划继续后置
