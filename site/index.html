1.6 亮点

  • TXT 首次打开改为复用解析缓存，并补齐 UTF-16 识别和无章节正文兜底。
  • EPUB 导入现在兼容“结构脏但可读”的文件，坏文件也会给出更明确的错误提示。
  • 阅读页、发现页、“我的”页和 API 中心一起做了可用性收口，日常使用路径更顺手。

下载与验证

  1. 正式包从 GitHub Releases 下载，最新版本固定指向 `release APK`。
  2. 这次 1.6 直接从 `main` 走云端签名发布，避免继续依赖本地 Android 打包环境。
  3. 本地即使没有 Java，也可以直接使用 GitHub Actions 云端跑校验与正式发版；release 仍以 `main` 上 tag 为准。
