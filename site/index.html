1.4 亮点

  • 书源 bridge 升级为结构化规则模型，统一支持 HTML/CSS、XPath、JSONPath 三条主链。
  • 新增 `##` 文本清洗、轻量 `@js:` 后处理与 unsupported 规则显式诊断，减少静默空结果。
  • 首批 10 个目标源 fixture、shared archetype 与 GitHub 云端验证闭环已补齐。

下载与验证

  1. 正式包从 GitHub Releases 下载，最新版本固定指向 `release APK`。
  2. 这次 1.4 先在 `publish/main` 跑通 `android-ci`，再快进同步到 `main`。
  3. 本地没有 Java 时，直接使用 GitHub Actions 云端跑 `testDebugUnitTest + assembleDebug`；正式发版仍以 `main` 上 tag 为准。
