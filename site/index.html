1.3 亮点

  • 发现页搜索与选书结果改为后台线程执行，网文搜索和加入流程不再拖住主线程。
  • TXT / EPUB 阅读进度改为基于全文段落数推导，顶部进度条、文案与目录状态实时同步。
  • 发布链路新增发布页版本校验，手动发版只接受当前 main 对应 tag，并切到 Node 24 运行时。

下载与验证

  1. 正式包从 GitHub Releases 下载，最新版本固定指向 `release APK`。
  2. 开发分支先走 `android-ci`，由 GitHub Actions 生成 `debug-apk` artifact。
  3. 本地没有 Java 时，直接使用 `gh run watch` 跟进云端构建状态；正式发版仍以 `main` 上已存在 tag 为准。
