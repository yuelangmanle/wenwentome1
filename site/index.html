1.5 亮点

  • TXT 新增 UTF-8 / GB18030 编码识别与章节切分修复，中文 TXT 兼容性更稳。
  • TXT 阅读同时兼容旧版全书段落进度和新版章节目录跳转，继续阅读不再错位。
  • EPUB 可读目录回退、详情元数据边界处理与非法导入拦截已经补齐。

下载与验证

  1. 正式包从 GitHub Releases 下载，最新版本固定指向 `release APK`。
  2. 这次 1.5 直接从 `main` 走云端签名发布，避免继续依赖本地 Android 打包环境。
  3. 本地即使没有 Java，也可以直接使用 GitHub Actions 云端跑校验与正式发版；release 仍以 `main` 上 tag 为准。
