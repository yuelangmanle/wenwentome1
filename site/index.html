1.2 亮点

  • API Hub 总览、Provider 管理、模型绑定、预算回退、价格目录与调用记录全面收口。
  • 阅读器 AI 面板支持章节总结、段落解释、翻译与 TTS 入口，结果缓存复用。
  • TXT / EPUB / WEB 回归，书架筛选排序、EPUB 优先正文与网文健康增强同步落地。

下载与验证

  1. 正式包从 GitHub Releases 下载，最新版本固定指向 `release APK`。
  2. 开发分支先走 `android-ci`，由 GitHub Actions 生成 `debug-apk` artifact。
  3. 本地没有 Java 时，直接使用 `gh run watch` 跟进云端构建状态。
